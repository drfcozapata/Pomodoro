# Pomodoro 🍅

Una aplicación de temporizador Pomodoro para macOS que te ayuda a mantener el enfoque y la productividad.

## Características

- **Ciclo de Pomodoro tradicional**: 4 sesiones de enfoque de 25 minutos + 3 descansos cortos de 5 minutos + 1 descanso largo de 15 minutos
- **Indicador visual**: Círculo que muestra el progreso del tiempo restante
- **Colores diferenciados**:
  - Enfoque: Verde
  - Descanso Corto: Rojo
  - Descanso Largo: Azul
- **Ventana de descanso**: Durante los períodos de descanso, la app ocupa toda la pantalla con fondo translúcido para notificarte que es hora de descansar
- **Minimización automática**: Después de un descanso, la app se minimiza al dock para volver al segundo plano
- **Configuración personalizable**: Ajusta los tiempos de enfoque y descanso según tus preferencias

## Uso

1. Haz clic en el botón de play para iniciar el temporizador
2. Durante el enfoque, trabaja en tu tarea
3. Cuando termine el enfoque, la app mostrará una pantalla completa indicándote que es hora de descansar
4. Después del descanso, la app se minimizará automáticamente al dock
5. Repite el ciclo

## Requisitos

- macOS 12.0 o superior

## Instalación

1. Clona el repositorio
2. Abre `Pomodoro.xcodeproj` en Xcode
3. Compila y ejecuta (Cmd + R)

## Configuración

Haz clic en el botón de "Ajustes" para cambiar:
- Tiempo de enfoque (minutos)
- Tiempo de descanso corto (minutos)
- Tiempo de descanso largo (minutos)
